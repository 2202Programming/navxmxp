This directory contains the following files:

navx-micro_board.skp:  Sketchup File for the navX-Micro Circuit Board
navx-micro_base.skp:  Sketchup File for the navX-Micro Enclosure Base (X1000 scale)
navx-micro_lid.skp:  Sketchup File for the navX-Micro Enclosure Lid (X1000 scale)

In addition, the _scaleddown.skip files are sketchup files at actual (millimeter) scale,
and the .stl files are Stereo-lithography format files for 3D printing, also at actual (millimeter) scale.
